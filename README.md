# unit_3_arrays

1. Написать консольную программу “Шахматный ход”
2. Программа должна предложить пользователю выбрать фигуру и ее
   цвет
3. Программа предлагает пользователю поставить фигуру в любое
   место на доске
4. Программа, после ввода пользователя положения фигуры, должна
   проверить, возможно ли это – если нет, повторяется пункт 2
5. Если да, программа предлагает пользователю переместить фигуру в
   любую точку доски.
6. Программа проверяет корректность координат, вводимых
   пользователем, если не валидны – предложение на ввод координат
   продолжается, если валидны – программа перемещает фигуру
7. После перемещения, программа предлагает вариант продолжения
   перемещения фигуры или перейти на пункт 2.
