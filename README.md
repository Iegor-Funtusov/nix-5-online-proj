# nix-5-online-proj
##Run program:
1. Run Service class from IDE
2. Run the run.bat file
### Available options for array: 

    0 - exit  
    1 - find even numbers  
    2 - count positive numbers  
    3 - count elements greater than previous  
    4 - count elements that have two smaller neighbors  
    5 - reverse order  
    6 - swap neighbors
